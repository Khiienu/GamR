In this clone of Flickr, Gamers(users) are allowed to post pictures of anything game related. 
They can create albums and add pictures from other users to that album . 

HOW TO RUN THE SERVER ON LOCAL HOST >
* Clone the repo
* access the backend
* npm install
* npm start

THEN >
* access the frontend 
* npm install 
* npm start 

The site should run!
